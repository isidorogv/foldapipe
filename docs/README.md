## Assembled Printer

* **Print volume (test model):** 140x140x135mm.

* **Print volume (maximun):** 150x150x150mm

* **Printer size (foldded):** (L)390x(W)150x(H)460mm.

* **Printer size (unfoldded):** (L)390x(W)410x(H)380mm.

* **Printer Weight:** 5 kg.

* **Pipe diameter:** 20mm.
